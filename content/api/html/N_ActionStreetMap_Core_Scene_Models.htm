ActionStreetMap.Core.Scene.Models Namespace
This namespace contains classes which represent map data primitives. So far, they are closely related to OSM primitives. You can find additional information about them here: http://wiki.openstreetmap.org/wiki/Elements.
Classes
  ClassDescription
Area
Represents connected polygon. Used for buildings, park areas, etc.
Canvas
Represents canvas (terrain).
Model
Represents abstract model which can be represented via OSM data.
Node
Represents Node - point on map with associated tags.
Relation
Represents relation between different models. This abstraction is introduced as we need to linke some models together (part of buildings, outline of buildings).
Tile
Represents map tile.
Way
Represents a set of connected points. For example, used for roads
