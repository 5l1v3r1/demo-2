This namespace contains essential Unity independed classes.
Namespaces
NamespaceDescription
ActionStreetMap.Core
This namespace contains essential core classes.
ActionStreetMap.Core.Elevation
This namespace contains classes which process elevation data.
ActionStreetMap.Core.Elevation.Srtm
This namespace contains classes to process SRTM elevation files. You can find additional info and download links here: http://www2.jpl.nasa.gov/srtm/
ActionStreetMap.Core.MapCss
This namespace contains classes to work with MapCSS format. You can read more here: http://wiki.openstreetmap.org/wiki/MapCSS
ActionStreetMap.Core.Positioning
This namespace contains classes to process geo position logic.
ActionStreetMap.Core.Positioning.Nmea
This namespace contains classes to read NMEA files. You can read more here: http://wiki.openstreetmap.org/wiki/NMEA
ActionStreetMap.Core.Scene
This namespace contains classes which represent or used to create/process scene models (e.g. building, roads, etc.).
ActionStreetMap.Core.Scene.Models
This namespace contains classes which represent map data primitives. So far, they are closely related to OSM primitives. You can find additional information about them here: http://wiki.openstreetmap.org/wiki/Elements.
ActionStreetMap.Core.Scene.World
This namespace contains classes which represent abstractions of real world (e.g. building, road, address, etc.)
ActionStreetMap.Core.Unity
This namespace contains classes which used to substitute Unity classes
ActionStreetMap.Core.Utilities
This namespace contains utility classes.
