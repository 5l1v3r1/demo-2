Demos

This page contains links to web player builds. Each build contains small map parts of different cities: Moscow, Berlin, New York. Scene is constructed dynamically by processing OSM data file and creating game objects. Please note, that web builds below doesn't contain any logic to improve rendered objects (e.g. custom terrain shaders, special materials, etc.). Therefore, the visual representation is quite ugly.

Discover Red Square, Moscow. Tile size: 3km x 3km

Discover Tiergarten, Berlin. Tile size: 0.5km x 0.5km

Discover Manhattan, New York. Tile size: 0.5km x 0.5km
